
# Aircraft Readiness Calculator

Python-based aircraft readiness calculator that analyzes operational status and system failure breakdown.


## Features

- Calculates aircraft readiness percentage
- Displays breakdown of failures by system
- Outputs total number of down aircraft
- Clean, formatted command-line output


## Purpose

The goal of this project is to bridge the gap between learning Python fundamentals and applying them to real-world data analytics problems.

Rather than working with generic datasets, this project is based on an aviation context to align with my domain experience within aviation and aircraft maintenance, and work towards long-term goals in defense/aircraft analytics.
## Future Improvements
